# 🚀 Space Shooter 太空射击游戏 - Python课程设计项目

## 一、项目简介
Space Shooter 是一款基于 Python 与 Pygame 库开发的 2D 太空射击小游戏，作为本次Python课程设计的实践成果，项目完整实现了经典街机射击游戏的核心玩法，包括玩家控制、敌方生成、碰撞检测、分数统计等功能。游戏以太空为背景，玩家操控飞船躲避敌方攻击并发射子弹消灭敌人，通过不断挑战获取更高分数，既锻炼了编程逻辑，也直观展现了面向对象编程、游戏循环机制等Python核心知识点的应用。

## 二、开发环境与技术栈
### 1. 开发环境
- 操作系统：Windows 11
- 开发工具：PyCharm Community Edition 2023.x
- 编程语言：Python 3.7
- 依赖库：Pygame 2.x

### 2. 核心技术点
- 游戏主循环：基于 Pygame 的事件循环机制，实现游戏的刷新、渲染与逻辑更新
- 键盘事件处理：通过 `pygame.key.get_pressed()` 实现飞船的移动控制与射击触发
- 精灵类（Sprite）管理：使用 Pygame 的 Sprite 类与 Group 类，高效管理玩家、敌人、子弹等游戏对象
- 碰撞检测：通过 `pygame.sprite.spritecollide()` 实现子弹与敌人、玩家与敌人的碰撞判定
- 图形与文本渲染：加载飞船、子弹、背景素材，实现游戏界面的绘制与分数文本的动态更新

## 三、游戏玩法说明
### 1. 操作方式
- 方向键「↑↓←→」：控制飞船上下左右移动，躲避敌方子弹与敌人本体
- 空格键：发射子弹，消灭前方出现的敌方目标
- 游戏结束后可重新运行程序再次挑战最高分

### 2. 游戏规则
- 玩家初始拥有1条生命值，被敌人或敌方子弹击中则游戏结束
- 每消灭一个敌人可获得10分，分数实时显示在游戏界面顶部
- 敌人会从屏幕顶部随机位置生成，并以固定速度向下移动，难度随游戏进行略有提升

## 四、核心模块实现解析
### 1. 玩家飞船模块
玩家飞船类继承自 `pygame.sprite.Sprite`，封装了飞船的位置、速度、生命值等属性，以及移动、射击、更新等方法。通过边界判定限制飞船的移动范围，避免飞船超出游戏窗口；射击功能通过生成子弹对象并加入子弹精灵组实现，同时控制射击频率，防止子弹无限制生成。

### 2. 敌方生成与移动模块
敌人对象同样使用精灵类实现，通过定时器事件实现敌人的周期性生成，保证游戏的节奏性。每个敌人拥有独立的移动速度，通过随机数实现敌人出现位置的多样化，增加游戏的随机性与挑战性。敌人到达屏幕底部或被玩家子弹击中后会自动销毁，避免内存占用。

### 3. 碰撞检测与分数系统
碰撞检测是游戏的核心逻辑之一，通过 `spritecollide()` 方法分别检测玩家子弹与敌人、玩家飞船与敌人/敌方子弹的碰撞。当子弹与敌人碰撞时，敌人对象被销毁，玩家分数增加；当玩家与敌人/敌方子弹碰撞时，玩家生命值归零，游戏结束并显示最终分数。

### 4. 游戏界面与状态管理
游戏界面包括背景图、玩家飞船、敌人、子弹、分数文本等元素，通过 `pygame.display.flip()` 实现每帧的刷新。游戏状态分为运行中、游戏结束两种状态，通过标志位控制不同状态下的逻辑执行，例如游戏结束时停止生成敌人、显示最终分数并等待用户重新开始。
